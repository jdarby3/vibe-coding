Pre-deploy checklist:

1. Run linter and fix any issues.
2. Run full test suite. Do not proceed if tests fail.
3. Check for any TODO/FIXME/HACK comments in changed files — resolve or document.
4. Verify environment variables are set for the target environment.
5. Confirm the build succeeds: run the project's build command.
6. Summarize what's being deployed: list changed files and features.
7. Ask the user for final confirmation before pushing/deploying.
