Ask the user the following questions one at a time, then write their answers to `.claude/rules/stack.md`.

Questions:
1. What language(s) will this project use? (e.g. TypeScript, Python, Go)
2. What framework or runtime? (e.g. Next.js, FastAPI, plain Node)
3. What database, if any? (e.g. Postgres, SQLite, none)
4. What styling approach? (e.g. Tailwind, CSS Modules, styled-components, none)
5. Any libraries that are already decided or strongly preferred?
6. Any patterns, libraries, or approaches to actively avoid?

After collecting all answers, write `.claude/rules/stack.md` with the answers filled in. Confirm with the user when done.
