Write a handoff note for the next session (human or AI). Save to `.claude/state/handoff.md`.

Include:
- **What was done** this session (bullet list, be specific).
- **Current state**: does it build? tests pass? anything broken?
- **What's next**: the immediate next task(s).
- **Blockers or open questions**: anything unresolved.
- **Key decisions made** and why.

Keep it under 50 lines. A future reader with no context should be able to pick up exactly where you left off.
